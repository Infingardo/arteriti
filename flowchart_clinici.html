Come inviare i dati clinici
Biopsia Arteria Temporale — GCA
Istruzioni passo-passo per il medico richiedente · Da tenere vicino al computer
1
Aprire il modulo dati clinici
Aprire il file richiesta_biopsia_temporale.html nel browser (Chrome, Firefox o Edge — non Internet Explorer).
Non trovate il file? Richiederlo alla SC Anatomia Patologica oppure scaricarlo qui.
2
Compilare tutti i campi del modulo
Compilare almeno i campi obbligatori. Più dati si inseriscono, più accurato sarà il referto.
!
Cognome, nome, data di nascita del paziente — verificare che corrispondano alla richiesta istologica
!
Steroidi: il paziente li ha già iniziati? Se sì → farmaco, dose e numero esatto di giorni prima del prelievo
!
Sintomi: cefalea, claudicatio mascellare, disturbi visivi (cecità, diplopia)
!
VES e PCR — inserire i valori numerici, non solo "elevata"
?
Halo sign ecografico — se l'eco-Doppler è stato eseguito
?
Sospetto clinico — alto / intermedio / altra diagnosi differenziale
3
Esportare il file
In fondo al modulo, cliccare il bottone:
⬇ Esporta dati clinici
Il browser scaricherà automaticamente un file con nome:
dati_clinici_GCA_COGNOME.json
Dove finisce il file? Nella cartella Download del computer. Controllare che il cognome nel nome del file sia quello corretto prima di procedere.
4
Inviare il file per e-mail all'Anatomia Patologica
Aprire il client di posta del reparto e inviare una nuova e-mail a:
✉
anatomia.patologica.fbf
@asst-fbf-sacco.it
Posta interna ospedaliera — non serve VPN
Oggetto e-mail (copiare questo schema)
Biopsia temporale GCA · COGNOME NOME · GG/MM/AAAA
Allegare il file JSON scaricato al punto 3.
Non incollare i dati nel corpo dell'e-mail — il file viene importato direttamente nel sistema di refertazione.
5
Inviare il campione biologico
Inviare il campione all'Anatomia Patologica con la richiesta istologica cartacea standard, compilata con gli stessi dati anagrafici del modulo.
🧪
Campione in formalina 10% · Etichettare con nome, cognome, data di nascita, data prelievo
Destinazione: SC Anatomia Patologica · Osp. Fatebenefratelli · P.le Principessa Clotilde 3
✓ Tutto fatto. Il patologo riceverà il file con i dati clinici e potrà importarli direttamente nel sistema di refertazione. Il referto sarà disponibile nei tempi abituali.
